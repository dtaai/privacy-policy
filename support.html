Common Issues

Can't generate routes?

• Check that location permissions are enabled
• Ensure you have an internet connection
• Try a different starting location

Routes aren't the right distance?

• The app generates routes as close as possible to your target distance
• Try adjusting the distance or choosing a different location

Location not working?

• Go to Settings → Privacy → Location Services
• Make sure Location Services is enabled
• Enable location access for the app
